## Режимы и Синтаксис Ввода

Ниже представлены правила ввода выражений для каждого типа калькулятора.

### 1. Basic Calculator (Базовый калькулятор)

Этот режим поддерживает стандартные арифметические операции.

Расчеты производятся по шаблону: **«число1» «оператор» «число2»**.

| Тип операции | Операторы | Примеры |
| :--- | :--- | :--- |
| **Бинарные** | `+`, `-`, `*`, `/` | `2 + 5`, `19 * 12`, `25 / 10` |

---

### 2. Scientific Calculator (Научный калькулятор)

Этот режим включает все базовые операции, а также тригонометрические функции и степень.

#### Стандартные расчеты

Расчеты производятся по шаблону: **«число1» «оператор» «число2»**.

| Операторы | Примеры |
| :--- | :--- |
| `+`, `-`, `*`, `/` | `10 + 2`, `5 * 5` |

#### Расчеты Sqrt, Sin, Cos

Расчеты производятся по шаблону: **«оператор» «число»**.

| Операторы | Примеры |
| :--- | :--- |
| `Sqrt`, `Sin`, `Cos` | `Sqrt 9`, `Sin 5`, `Cos 3` |

#### Расчет возведения в степень (Pow)

Расчет производится по шаблону: **«число» «оператор» «степень»**.

| Оператор | Примеры |
| :--- | :--- |
| `Pow` | `2 Pow 4` |

---

### 3. Programmer Calculator (Программистский калькулятор)

Этот режим предназначен для работы с целыми числами, побитовыми операциями и конвертацией систем счисления.

#### Стандартные расчеты

Расчеты производятся по шаблону: **«число1» «оператор» «число2»**.

| Операторы | Примеры |
| :--- | :--- |
| `+`, `-`, `*`, `/` | `10 + 5`, `20 / 4` |

#### Побитовые операции

Расчеты производятся по шаблону: **«число1» «оператор» «число2»**.

| Операторы | Примеры |
| :--- | :--- |
| `And` (`&`), `Or` (`|`), `XOr` (`^`) | `4 & 15`, `45 xor 102` |

> **Примечание:** В логических операциях в качестве оператора можно вводить как символ (`&`), так и полное слово (`And`).

#### Конвертация систем счисления

Расчеты производятся по шаблону: **«оператор» «число»**.

| Операторы | Примеры |
| :--- | :--- |
| `ToBinary`, `ToHex` | `toBinary 1024`, `toHex 92351` |
